# Watchlist App

A React application that displays products with search functionality and list virtualization for performance optimization.

## Features

* Fetch products from API
* Search products
* Debounced search
* List virtualization
* Performance optimization

## Tech Stack

* React
* JavaScript
* CSS
* Fetch API

## Concepts Practiced

* useEffect
* useMemo
* useRef
* Debouncing
* Binary Search
* List Virtualization
* Performance Optimization

## Future Improvements

* Infinite Scroll
* Dynamic Height Virtualization
* Better UI Design

## How to Run

```bash
npm install
npm run dev
```
